Amazon Sales Performance Dashboard
📌 Project Overview

This project is an interactive Power BI dashboard developed to analyze Amazon sales data. The dashboard helps identify sales trends, customer behavior, product performance, and profitability through data visualization.

The goal of this project is to demonstrate my data analysis skills by transforming raw sales data into meaningful business insights that support decision-making.

🎯 Objectives
Analyze overall sales performance.
Identify top-performing products and categories.
Monitor monthly and yearly sales trends.
Compare profit and revenue.
Understand customer purchasing behavior.
Build an interactive dashboard for business users.
🛠 Tools Used
Power BI

DAX
Data Cleaning

📊 Dashboard Features
Total Sales
Total Profit
Total Orders
Sales by Category
Sales by Region
Monthly Sales Trend
Top Selling Products
Interactive Filters (Year, Category, Region)
📈 Key Insights
Sales are concentrated in a few high-performing product categories.
Seasonal trends indicate higher sales during festive months.
Some products generate high revenue but relatively low profit.
Certain regions consistently outperform others in sales.
Interactive filters enable detailed analysis across multiple dimensions.
💡 Business Recommendations
Increase inventory for top-selling products.
Focus marketing campaigns on high-profit categories.
Improve performance in low-sales regions.
Optimize pricing for low-profit products.
Monitor seasonal demand to improve inventory planning.
